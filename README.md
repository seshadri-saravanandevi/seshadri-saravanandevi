## 🚀 Featured Projects

### 🤖 [ROS2 Vision-Language Pick-and-Place](https://github.com/seshadri-saravanandevi/seshadri_saravanandevi-ros2-vlm-pickplace)
A ROS2 pipeline where a typed command ("pick up the red block") is grounded by OWL-ViT, an open-vocabulary vision-language model, and executed by a simulated UR5e arm via MoveIt2 motion planning in Gazebo.

`ROS2` `Python` `MoveIt2` `Gazebo` `OWL-ViT` `PyTorch`

### 🚗 [BMS Simulation with UDS Diagnostics over SocketCAN](https://github.com/seshadri-saravanandevi/bms-can-diagnostics)
A multi-cell EV battery management system — SOC estimation via coulomb counting, fault detection, cell balancing logic — exposing live data through real UDS (ISO 14229) diagnostic services (ReadDataByIdentifier, ReadDTCInformation, ECUReset) over genuine Linux SocketCAN.

`CAN` `UDS` `Python` `SocketCAN`

### ⚙️ [FreeRTOS Body Control Module](https://github.com/seshadri-saravanandevi/freertos-bcm-simulation)
A simulated automotive BCM running real FreeRTOS: five concurrent tasks coordinated with mutexes, dual event queues, and a software-timer-driven watchdog task, reporting live status over SocketCAN.

`FreeRTOS` `Embedded C` `Real-Time Systems` `SocketCAN`
